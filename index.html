Welcome to Meet Ortansia and MORE!

About Ortansia

Ortansia is an Image acquisition software designed for optical fluorescent bio-imaging studies. The software is developed with Python 3 and LabVIEW 2018 (or later). The package is distributed as both individual components and integrated asquisition suite.

Learn More

About Kyma

Kyma is a Python coded software designed for Optical Point Spread Function Engineering with the implementation of GPU accelerating algorithm. GUI is provided for a user-friendly experience. The software is designed for all kinds of Platforms (Win/Mac/Linux) and for all kinds of GPUs (Nvidia/AMD).

Learn More

About DavinSci

DavinSci is an image processing program designed for scientific image processing. The special functions of this program help researchers to easily analyse the image sequences and stacks. The program is provided with both individual components and integrated GUI.
